Project that provides a "localhost" ActiveMQ and Derby Database Server instance.

IMPORTANT: Start this before starting the Command & Query Server!

START using "main(..)" method:
org.fuin.auction.infrastructure.InfrastructureStart

SHUTDOWN using "main(..)" method:
org.fuin.auction.infrastructure.InfrastructureStop

If you use Eclipse you can use the predefined launch configurations in this directory.
